# robot_hand-urdf
The URDF file of robotic arm (UR5e, UR10, etc.) connected with robotic hand (LEAP Hand, etc.)

- **ur5e_leaphand.urdf**: the URDF file of [UR5e](https://www.universal-robots.com/products/ur5e/) connected with [LEAP Hand](https://v1.leaphand.com/). The URDF file can be previewed on the website: https://viewer.robotsfan.com/
- **ur10_leaphand.urdf**: the URDF file of UR10 connected with LEAP Hand.
